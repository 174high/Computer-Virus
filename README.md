## Computer-Virus

这里是我个人开发的一些电脑病毒以及相应的专杀工具，欢迎各位star && fork，为了打造一个学习编写病毒的大家庭而不懈努力。

目前已经写了如下病毒：

- panda
- BlueSpider
- EternalBlue
